Event Description

Good design is all about making other designers feel like idiots because that idea wasn’t their. Php, html, css and javascrpit! Languages that you love. If words are your world, then come whip the developer in you!

Rules 

Round 1

  • Elimination round. This round consists of 30 MCQ questions which have to be answered in 25 minutes. The questions are based on web programming languages.

Round 2

  • Information will be given about website looks i.e. guidelines but NOT images.
  • The website should have : Functionality, Attractive animation and should be eye catching,
  • Interactive/Responsive.
  • The participants can register either individually or in a team of 2 members.
  • Marks will be given on basis of: Validation, Responsiveness, Security etc.
  • Decision of judges will be final and abiding.
  • Round 2 is a final round. This round is a 120 minutes round in which the participants are required to make a website on a theme which will be provided to them just after round 1 if they have already qualified round 1.The participants are allowed to use following web languages- HTML, CSS, PHP, JAVASCRIPT, JQUERY.
  • They all are provided with-MYSQL, APACHE with PHP, TEXT EDITOR(sublimetext), INTERNET connection, LINUX system(Ubuntu)/Windows and Simple Photo Editor(gimp).
