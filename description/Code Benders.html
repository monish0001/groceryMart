Event Description

This will be an online overnight coding contest hosted on CodeChef, of three hours duration. This is not a team event and there won’t be any separate rounds, just one main coding round, that’s it.

There are no restrictions on participation, so any individual can participate. One just needs to have a CodeChef account which can be created here – https://www.codechef.com/signup.

(One can also login using Google, Facebook or GitHub account)

Prize money will only be given to the top Indian performer .

Rules

  • There will be 5 – 6 problems (of varying difficulty levels) which will be logical, mathematical and algorithmic in nature.
  • Participants can write solution to a problem in any programming language of their choice. Their code should produce the correct answer and adhere to the constraints given for a problem, in order to get an ACCEPTED verdict.
  • Contest format will be similar to IOI, and so will be the rank list style (IOI Style (https://www.codechef.com/rankings#IOIStyleRanklist)).
  • The result will be declared promptly after the contest, or on the next day.
  • Decisions made by the event organizers will be final and binding. Any arguments or nuisance will not be tolerated.
  • Start Time 09:00 PM (2nd February 2019)
  • End Time 12:00 AM (3rd February 2019)
  • Duration 3 Hours
